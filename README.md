# Diabetes Prediction

Machine learning project for diabetes prediction using medical data.

## 📋 Requirements

- Python 3.7+
- pandas
- numpy
- scikit-learn
- matplotlib

## 🚀 Quick Start

```bash
python Diabet.py
```

## 🔧 ML Tools

This project uses tools from [TOOL-BOX](https://github.com/Arefmtl/TOOL-BOX) — a comprehensive ML toolbox with 8 specialized modular tools.
